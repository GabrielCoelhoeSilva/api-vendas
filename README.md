## 📋 Sobre o Projeto

A **API de Vendas** é uma aplicação backend desenvolvida com **Spring Boot**, projetada para servir como base de um sistema de vendas robusto e escalável. Ela centraliza o gerenciamento de **clientes** e **produtos**, oferecendo uma fundação sólida para futuras integrações com módulos de pedidos, pagamentos e relatórios.

## 🏗️ Estrutura do Projeto

O projeto segue a estrutura padrão de aplicações Spring Boot, organizado em módulos dedicados às principais entidades de negócio:

```
src/main/java/com/seuprojeto/apivendas/
├── cliente/      # Lógica e recursos de gestão de clientes
├── produto/      # Lógica e recursos de gestão de produtos
└── ...
```

- **`cliente`** — Cadastro, consulta, atualização e remoção de clientes.
- **`produto`** — Cadastro, consulta, atualização e remoção de produtos.
